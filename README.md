# ErrorMux

A zsh plugin for on-demand shell command error explanations.

## Installation

```bash
# Install dependencies
uv sync

# Run the CLI
uv run errormux explain
```

## Usage

When a command fails in your zsh shell, type `??` to get an explanation.

## Requirements

- zsh 5.9+
- Python 3.12+
- Ollama with gemma3:4b model
