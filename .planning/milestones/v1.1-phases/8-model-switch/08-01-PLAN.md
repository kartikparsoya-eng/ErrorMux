# Plan 08-01: Model Constant and Cache Key Update

**Wave:** 1 (autonomous)

## Objective

Update the Ollama model constant and modify cache key generation to include model name.

## Tasks

### Task 1: Update Model Constant

**File:** `src/errormux/client.py`

**Change:** Line 10
```python
# Before
OLLAMA_MODEL = "gemma3:4b"

# After
OLLAMA_MODEL = "gemma4:e2b"
```

---

### Task 2: Add Cache Clear Function

**File:** `src/errormux/cache.py`

**Add after `cache_delete()` function:**
```python
def cache_clear() -> None:
    """Clear all cached responses."""
    conn = sqlite3.connect(CACHE_DB)
    conn.execute("DELETE FROM cache")
    conn.commit()
    conn.close()
```

---

### Task 3: Update Cache Key to Include Model

**File:** `src/errormux/cache.py`

**Change:** Line 13-15
```python
# Before
def make_cache_key(cmd: str, stderr: str) -> str:
    """Generate SHA256 cache key from command and stderr."""
    return hashlib.sha256(f"{cmd}|{stderr}".encode()).hexdigest()

# After
def make_cache_key(model: str, cmd: str, stderr: str) -> str:
    """Generate SHA256 cache key from model, command, and stderr."""
    return hashlib.sha256(f"{model}|{cmd}|{stderr}".encode()).hexdigest()
```

---

### Task 4: Update CLI Caller to Pass Model

**File:** `src/errormux/cli.py`

**Find:** All calls to `make_cache_key()`

**Change:** Add model parameter
```python
# Before
key = make_cache_key(cmd, stderr)

# After
key = make_cache_key(OLLAMA_MODEL, cmd, stderr)
```

**Note:** Import `OLLAMA_MODEL` from client module if not already imported.
