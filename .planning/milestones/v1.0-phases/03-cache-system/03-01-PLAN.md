## Task 03-01-01: Create cache module with key generation

<read_first>
- `src/errormux/cli.py` — understand current CLI structure
- `pyproject.toml` — check existing dependencies
- `.planning/phases/03-cache-system/03-CONTEXT.md` — D-02 minimal schema decision
</read_first>

<action>
Create `src/errormux/cache.py` with:

1. Import statements:
   - `import hashlib` for SHA256
   - `import sqlite3` for database
   - `import time` for TTL
   - `from pathlib import Path`

2. Constants:
   - `CACHE_DIR = Path.home() / ".shell-explainer"`
   - `CACHE_DB = CACHE_DIR / "cache.db"`
   - `TTL_SECONDS = 7 * 24 * 60 * 60` (7 days)

3. Function `make_cache_key(cmd: str, stderr: str) -> str`:
   - Return `hashlib.sha256(f"{cmd}|{stderr}".encode()).hexdigest()`

4. Function `_ensure_db()`:
   - Create CACHE_DIR if not exists: `CACHE_DIR.mkdir(parents=True, exist_ok=True)`
   - Create table if not exists:
     ```sql
     CREATE TABLE IF NOT EXISTS cache (
         key TEXT PRIMARY KEY,
         response TEXT NOT NULL,
         created_at REAL NOT NULL
     )
     ```
   - Create index: `CREATE INDEX IF NOT EXISTS idx_created_at ON cache(created_at)`

5. Function `cache_get(key: str) -> str | None`:
   - Call `_ensure_db()`
   - Connect to CACHE_DB
   - Query: `SELECT response, created_at FROM cache WHERE key = ?`
   - If no row: return None
   - If row exists and `time.time() - created_at > TTL_SECONDS`: delete row, return None
   - Else: return response

6. Function `cache_set(key: str, response: str) -> None`:
   - Call `_ensure_db()`
   - Connect to CACHE_DB
   - Insert or replace: `INSERT OR REPLACE INTO cache (key, response, created_at) VALUES (?, ?, ?)`
   - Use `time.time()` for created_at
</action>

<acceptance_criteria>
- `src/errormux/cache.py` exists
- File contains `def make_cache_key(cmd: str, stderr: str) -> str:`
- File contains `def cache_get(key: str) -> str | None:`
- File contains `def cache_set(key: str, response: str) -> None:`
- File contains `CACHE_DB = Path.home() / ".shell-explainer" / "cache.db"`
- File contains `TTL_SECONDS = 7 * 24 * 60 * 60`
</acceptance_criteria>
