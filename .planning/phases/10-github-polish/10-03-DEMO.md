## Objective

Create a demonstration GIF showing ErrorMux in action.

- Output: demo.gif in repository root
- Requires manual recording

## Tasks

### Task 1: Document demo recording process

**Type:** auto

**Action:** Create DEMO-RECORDING.md with instructions

**Content:** Steps to install terminalizer and record demo

### Task 2: Record demo GIF

**Type:** checkpoint:human-action

**Action:** User must manually record the demo

**Steps:**
1. Install terminalizer: `npm install -g terminalizer`
2. Record demo: `terminalizer record demo -k`
3. Render GIF: `terminalizer render demo -o demo.gif`
4. Move to repository root

**What to record:**
- Start a fresh terminal
- Run a command that fails (e.g., `ls /nonexistent`)
- Type `??` to trigger explanation
- Show the explanation output
- Press Enter to return to clean prompt
